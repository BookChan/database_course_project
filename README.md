# database_course_project
数据库课程项目，立题为学生选课系统。通过动手，体会数据库的魅力。


## 相关技术

- 服务器框架: python,Django
- 前端框架: bootstrap,jQuery
- 数据库: MySQL


## 主要工作
- 数据库的关系表设计
	- 纸上绘表
	- 初始化表
	- 数据获取
	- 数据导入

- 客户端到服务器端的通信设计
	- 验证信息
	- 记录信息
	- 数据传输
	- 安全，负载等测试

- 后台程序设计
	- 根据所有客户端请求可能应变
	- 日志记录
	- 获取信息
	- 处理信息
	- 返回信息

- 前端网页设计
 分多少块，块长什么样，放在哪里,会怎么变
	- html模板
	- 网页布局
	- 页面样式
	- 页面交互
